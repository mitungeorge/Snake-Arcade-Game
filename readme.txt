SNAKE GAME

Description: This is a simple classic arcade game,”SNAKE”,in which a player controls a snake, guiding it towards food and away from walls or itself. The game listen to the arrow keys, and the snake grows as it consume food. When the snake collides with itself or a wall, the game ends.

Build/Usage: Open the snake.html file in any browser and start playing.

Technologies Used: HTML5 ,CSS3 and Javascript
